# Football Data Analyzer

## Description

Ce projet vise à analyser les données des championnats de la Premier League et de la Liga
afin de produire des statistiques exploitables pour l’analyse prédictive des performances.
Il permet d’automatiser le traitement des données sportives pour faciliter la prise de décision.

## Technologies utilisées

- Python
- Pandas

## Crédits

Les données utilisées dans ce projet proviennent de sources open source spécialisées
dans les statistiques de football.
